<<Graphics 폴더 설명서>>
-----폴더
1. effect : 각종 이펙트

2. object : 맵 내의 오브젝트(책상, 의자, 칠판, 창문, 화분 등)
- object_(이름) 의 형식을 사용합니다.

3. tilemap : 맵 배경 타일맵(바닥, 벽, 외벽 등)
- tilemap_floor_(장소이름) = 바닥 타일맵
- tilemap_outline_(장소이름) = 외벽 타일맵
- tilemap_wall_(장소이름) = 내벽 타일맵

-----PDF
1. unity_PixelSprite 
- 유니티에 픽셀/도트 스프라이트 적용하는 방법을 담은 pdf 파일입니다.
- 픽셀 깨짐 현상, 색 뭉개짐 현상 해결 방법이 담겨 있습니다.
